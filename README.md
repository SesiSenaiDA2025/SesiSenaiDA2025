- **Sprint 4:** Desenvolvimento Mobile com Python Flet

- **Frontend Responsivo:** Bootstrap + CSS moderno

## 🛠️ Como Contribuir

1. **Clone** o repositório desejado
   ```bash
   git clone https://github.com/SesiSenaiDA2025/[nome-do-repo].git
   ```

2. **Crie** sua branch de desenvolvimento
   ```bash
   git checkout -b feature/minha-contribuicao
   ```

3. **Desenvolva** seguindo os padrões da turma
   ```bash
   # Estrutura de commits clara
   git commit -m "✨ feat: adiciona nova funcionalidade"
   git commit -m "🐛 fix: corrige bug no sistema"
   git commit -m "📚 docs: atualiza documentação"
   ```

4. **Envie** suas alterações
   ```bash
   git push origin feature/minha-contribuicao
   ```

5. **Abra** um Pull Request
